# 🪐 Chair Shop Landing Page (HTML/CSS)

## Description

Ceci est une landing page responsive pour un shop de chaises, réalisée dans le cadre de la formation Udemy
[udmey HTML CSS cours](https://www.udemy.com/share/101Wqo3@Af5rU4ak7OIVxanBQXEv8ZaBqTxPe_Il4wJKf-2cKL1WFKM-leK62SWK88GrLaz2Yg==/)

### La page présente :

- Une section héro avec un call-to-action “Shop chairs”
- Une section features présentant les avantages des chaises
- Une section testimonials avec avis clients
- Une section produits bestsellers avec images, détails et prix

Toutes les sections sont construites à la main en HTML et CSS, utilisant Grid et Flexbox pour la mise en page.

## Voir le projet en ligne

Tu peux consulter le projet en direct ici : [Chair Shop Landing Page](https://cynhlfn.github.io/Chair-shop-landing-page-HTML-CSS/)

## Technologies utilisé

- HTML5
- CSS3
- Google Fonts (Inter)
- Icônes SVG

## Compétences developpées :

- HTML sémantique et structuré
- CSS moderne : Flexbox, Grid, variables et pseudo-classes
- Typographie et système de couleurs
- Mise en page responsive
- Gestion des boutons et éléments interactifs
